**Weather App**
A responsive weather forecasting web app that provides 5-3 hourly forecast for any city. The app works seamlessly on both mobile and desktop devices.

**Features**
Get a 5-3 hourly weather forecast by entering a city name
Responsive design for mobile and desktop
Interactive and easy-to-use interface
Showcases front-end development skills and best practices
Dark/light mode toggle for better UX
Display more detailed weather info (temperature, weather description, chart)

**Technologies / Skills Used**
Front-end development
Responsive UI design
Dynamic DOM manipulation
API integration (Weather API)
Interactive and modern web design
